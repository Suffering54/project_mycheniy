# Прогулка Пушинки
### Правила игры
Пушинка и Пушин гуляли по лугу, наслаждаясь ароматом цветов. Вдруг, из неоткуда возник лабиринт. Это злой волшебник решил разлучить котеек. Но котики не растерялись и начали искать путь к друг другу. Тут злодей решил приковать Пушина к земле, чтобы пара не смогла встретиться, но он не учел того, что Пушинка любила подобного рода задачки и довольно быстро проходила лабиринт пока ей на пути не встретились черные дыры. Итак, котик замедлился. Помогите же котикам встретиться лицом к лицу. Проведите Пушинку по лабиринту мимо черных дыр. Пара надеется на тебя!
